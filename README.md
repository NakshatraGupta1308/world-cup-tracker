# World Cup 2026 Live Tracker Dashboard

A real-time, full-stack dashboard for the FIFA World Cup 2026 (June 11 - July 19, 2026). Track live standings, match scores, interactive bracket progression, team and player statistics, and live match discussions.

## Tech Stack

- Frontend: React 18 + Vite + Tailwind CSS
- Backend: Spring Boot 3 + WebSocket
- Database: PostgreSQL
- Deployment: Vercel (frontend) + Render (backend)

## Features

- Real-time group stage standings (12 groups)
- Full 104-match schedule with live scores
- Interactive knockout bracket that updates as the tournament progresses
- Team profiles with squad rosters
- Individual player statistics
- Head-to-head team comparisons
- Live chat per match (WebSocket)
- Responsive mobile-first design

## Getting Started

### Frontend
```bash
cd frontend
npm install
npm run dev
```

### Backend
```bash
cd backend
mvn spring-boot:run
```
